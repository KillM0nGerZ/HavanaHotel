# Kavana Hotel.
### Kavana Hotel  เป็นโปรแกรมจองโรงแรม

> ## วัตถุประสงค์ 
- เพื่อให้ผู้ใช้งานคิดคำนวณวันเวลา ราคา จำนวนห้อง การจองได้รวดเร็วขึ้น
